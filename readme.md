Rust Project for OKEX Market Maker

Env: Windows 11

-- Init Porject --
1. Follow https://www.rust-lang.org/learn/get-started to install rust
2. Download VScode, install 4 plugin
    * rust-analyzer\
    * Even Better TOML
    * Error Lens
    * CodeLLDB
3. cargo new OKEX_ALGO
4. Start build project
